The package includes an `_res` folder. Settings, clipboard history, and WebView2 internal data are stored there.

Highlights

The design keeps a plain-text editor feel while focusing on clipboard workflow and Markdown readability.

Clipboard capture tab Copied text is collected with timestamps and remains directly editable.
Improved Markdown readability Headings, rules, code fences, links, and tables are styled with lightweight decorations for better scanning.
Portable workflow Runs without installation and keeps settings plus history inside the `_res` folder.

Core Features

The feature set starts with core text editing and adds practical helpers around Markdown and clipboard work.

Tabbed editor Open multiple documents at once and create new tabs from the trailing `+` button.
Find / Replace Use in-page find and replace through the top-right overlay.
Markdown table formatting Columns are aligned with full-width character handling so tables stay readable.
