Key Features

Card and Grid Views Switch between detailed card view and compact grid view to manage multiple monitors.
Brightness & Contrast Adjust the brightness and contrast of each monitor intuitively using sliders.
Input Source Switching Directly switch active input sources like HDMI or DisplayPort from a dropdown menu.
Hotkeys and Auto-hide Instantly show the app using global hotkeys and optionally auto-hide it to the screen edge.
